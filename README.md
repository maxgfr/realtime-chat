# realtime-redis

realtime-redis is a realtime chatroom working with Redis, Express, Socket.io and IBM Watson Tone Analyser.

## Commands

```bash
# to build and run it :
docker-compose up --build
# to stop and delete it :
docker-compose down --rmi 'all'
```
